
# ⌨️ PhonoScribe

**PhonoScribe** is a shortcut-based phonetics app that enables you to write all the different symbols used in IPA. It was developed in python and it was made to bridge the gap between academic transcription and digital technologies in a university known as **UNCO**. It is both conceived for translator and teaching courses of English. It provides a seamless interface for capturing, remapping, and notifying users of complex keyboard inputs.

---

## 🚀 Key Features

### 1. **Intelligent Transcription Engine**
* **Symbol Mapping:** Effortlessly converts key presses into specialized numerical notations or phonetic symbols.
* **System-Level Overrides:** Employs low-level keyboard hooks to intercept and remap standard keys (such as `F1` functions) while suppressing default OS behaviors.

### 2. **Professional UI & UX**
* **Thread-Safe Popups:** Custom-built Tkinter notification windows designed to safely communicate with background threads.
* **System Tray Integration:** Runs quietly in the background via `pystray`.

---

## 🛠️ Technical Architecture

### **Built With**
* **Python 3.13+**
* **Tkinter:** High-performance, low-latency GUI framework.
* **Pillow (PIL):** Anti-aliased image processing and dynamic resizing.
* **Keyboard Library:** Global low-level hooks and key suppression.
* **Pystray:** Persistent system tray management.
